# Teddy Chat 사용 가이드

Teddy Chat에 오신 것을 환영합니다! 이 가이드는 채팅 인터페이스를 효과적으로 사용하는 방법을 안내합니다.

## 📝 시작하기

### 대화 시작하기

1. **직접 입력**: 화면 하단의 입력창에 메시지를 입력하고 전송 버튼을 클릭하거나 Enter 키를 누르세요.
2. **대화 시작 예시 사용**: 첫 화면에 표시되는 대화 시작 예시 버튼을 클릭하면 자동으로 해당 질문이 전송됩니다.

### 키보드 단축키

- `Enter`: 메시지 전송
- `Shift + Enter`: 줄바꿈
- `Cmd/Ctrl + Enter`: 편집 모드에서 메시지 전송

## 🎨 주요 기능

### 파일 업로드

- 📎 클립 아이콘을 클릭하여 이미지나 PDF 파일을 업로드할 수 있습니다.
- 드래그 앤 드롭으로도 파일을 첨부할 수 있습니다.
- 지원 형식: JPEG, PNG, GIF, WebP, PDF

### 도구 호출 표시/숨기기

- 🔧 렌치 아이콘을 클릭하여 AI가 사용하는 도구 호출을 표시하거나 숨길 수 있습니다.
- 도구 호출을 숨기면 최종 결과만 깔끔하게 볼 수 있습니다.

### 대화 기록 관리

- 왼쪽 사이드바에서 이전 대화 기록을 확인할 수 있습니다.
- 각 대화는 자동으로 제목이 생성되며, 클릭하면 해당 대화로 이동합니다.
- 대화 제목은 편집할 수 있으며, 필요 없는 대화는 삭제할 수 있습니다.

## ⚙️ 설정

화면 우측 하단의 설정 버튼을 통해 다음 항목을 변경할 수 있습니다:

### 테마 설정

- **라이트/다크 모드**: 밝은 테마와 어두운 테마 중 선택
- **자동 모드**: 시스템 설정에 따라 자동으로 변경

### 글꼴 설정

- **Sans Serif**: 기본 sans-serif 글꼴
- **Serif**: 읽기에 편한 serif 글꼴
- **Monospace**: 코드 작성에 적합한 고정폭 글꼴

### 글자 크기

- 작게, 중간, 크게 중에서 선택

### 채팅 너비

- **기본**: 읽기 편한 중간 너비
- **넓게**: 더 많은 정보를 한 번에 볼 수 있는 넓은 너비

## 💡 유용한 팁

1. **긴 답변 재생성**: AI의 답변이 마음에 들지 않으면 "재생성" 버튼을 클릭하여 새로운 답변을 받을 수 있습니다.

2. **컨텍스트 유지**: 같은 대화 내에서는 이전 메시지의 맥락이 유지됩니다. 연속된 질문을 할 때 유용합니다.

3. **코드 하이라이팅**: 코드 블록은 자동으로 구문 강조가 적용되어 읽기 편합니다.

4. **마크다운 지원**: 텍스트 서식, 목록, 링크 등 다양한 마크다운 문법이 지원됩니다.

5. **수식 렌더링**: LaTeX 문법을 사용하여 수학 공식을 표현할 수 있습니다.

## 🔍 문제 해결

### 메시지가 전송되지 않아요

- 인터넷 연결을 확인해주세요.
- 페이지를 새로고침하고 다시 시도해보세요.

### 파일이 업로드되지 않아요

- 파일 크기가 너무 큰지 확인해주세요.
- 지원하는 형식(JPEG, PNG, GIF, WebP, PDF)인지 확인해주세요.

### 대화 기록이 보이지 않아요

- 왼쪽 상단의 사이드바 토글 버튼을 클릭하여 사이드바를 열어보세요.
- 설정에서 대화 기록 표시가 활성화되어 있는지 확인해주세요.
