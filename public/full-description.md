# Agent Chat UI 사용 가이드

Agent Chat UI에 오신 것을 환영합니다. 이 가이드는 채팅 인터페이스의 모든 기능을 효과적으로 활용하는 방법을 안내합니다.

## 🚀 시작하기

### 대화 시작 방법

**직접 입력**

화면 하단의 입력창에 질문이나 메시지를 입력한 후 전송 버튼을 클릭하거나 Enter 키를 누릅니다.

**대화 시작 프롬프트 사용**

첫 화면에 표시되는 대화 시작 예시 버튼을 클릭하면 해당 질문이 자동으로 전송됩니다. 이는 AI와의 대화를 빠르게 시작할 수 있는 편리한 방법입니다.

### ⌨️ 키보드 단축키

- `Enter`: 메시지 전송
- `Shift + Enter`: 줄바꿈 입력
- `Cmd/Ctrl + Enter`: 편집 모드에서 메시지 전송

## ✨ 주요 기능

### 📎 파일 업로드

채팅에 이미지나 PDF 문서를 첨부할 수 있습니다.

**업로드 방법**

- 입력창 왼쪽의 클립 아이콘을 클릭하여 파일 선택
- 파일을 채팅 화면으로 드래그 앤 드롭
- 클립보드에서 이미지를 복사하여 붙여넣기

**지원 파일 형식**

- 이미지: JPEG, PNG, GIF, WebP
- 문서: PDF

### 🔧 도구 호출 표시 제어

AI는 답변을 생성하는 과정에서 다양한 도구를 사용합니다. 렌치 아이콘을 클릭하면 이러한 도구 호출 과정을 표시하거나 숨길 수 있습니다.

**표시 모드**: AI의 작업 과정을 상세히 확인할 수 있습니다.
**숨김 모드**: 최종 결과만 깔끔하게 표시됩니다.

### 📚 대화 기록 관리

**사이드바 액세스**

왼쪽 상단의 토글 버튼을 클릭하면 대화 기록 사이드바가 열립니다. 여기서 이전 대화 목록을 확인하고 원하는 대화를 선택할 수 있습니다.

**대화 제목**

각 대화는 자동으로 생성된 제목을 가지며, 제목을 클릭하여 편집할 수 있습니다. 의미 있는 제목을 설정하면 나중에 대화를 쉽게 찾을 수 있습니다.

**대화 삭제**

더 이상 필요하지 않은 대화는 삭제할 수 있습니다. 각 대화 항목에서 삭제 옵션을 선택하면 됩니다.

### 🔄 응답 재생성

AI의 답변이 만족스럽지 않은 경우, 메시지 하단의 재생성 버튼을 클릭하여 새로운 답변을 받을 수 있습니다. AI는 같은 질문에 대해 다른 관점이나 표현으로 답변을 생성합니다.

## ⚙️ 설정

화면 우측 하단의 설정 버튼을 통해 다양한 옵션을 변경할 수 있습니다.

### 🎨 외관 설정

**색상 테마**

- 라이트 모드: 밝은 배경의 테마
- 다크 모드: 어두운 배경의 테마
- 자동 모드: 시스템 설정에 따라 자동 전환

**글꼴 스타일**

- Sans Serif: 깔끔한 기본 글꼴
- Serif: 읽기 편한 세리프 글꼴
- Monospace: 코드 작성에 적합한 고정폭 글꼴

**글자 크기**

작게, 중간, 크게 중에서 선택할 수 있습니다. 가독성에 맞게 조정하세요.

### 💡 UI 동작 설정

**채팅 너비**

- 기본: 읽기 편한 중간 너비 (768px)
- 넓게: 더 많은 정보를 한 번에 볼 수 있는 넓은 너비 (1280px)

**도구 호출 자동 접기**

활성화하면 AI의 응답이 완료된 후 도구 호출 세부사항이 자동으로 접혀 대화 기록이 깔끔하게 유지됩니다.

## 🎯 고급 활용법

### 컨텍스트 유지

같은 대화 내에서는 이전 메시지의 맥락이 자동으로 유지됩니다. 따라서 연속된 질문을 할 때 매번 전체 배경을 설명할 필요가 없습니다.

**예시**

```
사용자: "파이썬으로 간단한 계산기를 만들어줘"
AI: [코드 제공]
사용자: "여기에 제곱근 기능도 추가해줘"
AI: [업데이트된 코드 제공]
```

### 📝 마크다운 지원

AI의 응답은 마크다운 형식으로 렌더링됩니다. 다음과 같은 요소가 지원됩니다:

- **제목**: 계층적 구조
- **목록**: 순서 있는 목록, 순서 없는 목록
- **링크**: 클릭 가능한 하이퍼링크
- **코드 블록**: 구문 강조가 적용된 코드
- **표**: 정돈된 데이터 표시

### 💻 코드 하이라이팅

코드 블록은 프로그래밍 언어에 따라 자동으로 구문 강조가 적용됩니다. 코드 블록 우측 상단의 복사 버튼을 클릭하면 코드를 클립보드에 복사할 수 있습니다.

### 📐 수식 렌더링

LaTeX 문법을 사용한 수학 공식이 깔끔하게 렌더링됩니다. 복잡한 수학적 표현도 명확하게 표시됩니다.

## 🔍 문제 해결

### 메시지가 전송되지 않는 경우

1. 인터넷 연결 상태를 확인하세요
2. 브라우저를 새로고침하고 다시 시도하세요
3. 문제가 지속되면 개발자 도구의 콘솔을 확인하세요

### 파일 업로드 실패

1. 파일 크기가 제한을 초과하지 않았는지 확인하세요
2. 지원되는 파일 형식(JPEG, PNG, GIF, WebP, PDF)인지 확인하세요
3. 브라우저의 파일 접근 권한이 허용되어 있는지 확인하세요

### 대화 기록이 보이지 않는 경우

1. 왼쪽 상단의 사이드바 토글 버튼을 클릭하여 사이드바를 여세요
2. 설정에서 대화 기록 표시 옵션이 활성화되어 있는지 확인하세요

### 설정 변경이 적용되지 않는 경우

브라우저를 강력 새로고침(Ctrl+Shift+R 또는 Cmd+Shift+R)하여 캐시를 지우고 다시 시도하세요.

## 💡 유용한 팁

### 명확한 질문하기

구체적이고 명확한 질문을 할수록 더 정확한 답변을 받을 수 있습니다. 필요한 경우 배경 정보나 제약사항을 함께 제공하세요.

### 대화 나누기

서로 다른 주제는 새로운 대화로 시작하는 것이 좋습니다. 이렇게 하면 각 대화의 컨텍스트가 명확하게 유지됩니다.

### 정기적인 정리

더 이상 필요하지 않은 대화는 삭제하여 대화 목록을 깔끔하게 유지하세요.
